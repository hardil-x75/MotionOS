# Security

## Supported Version

X75 MotionOS `0.1.0-beta` is the current public beta.

## Reporting Security Issues

Please report security or privacy issues privately:

x75labs@gmail.com

Include a clear description of the issue, steps to reproduce it, and any relevant screenshots or logs. Please avoid publishing security issues publicly until there has been reasonable time to investigate and respond.

## Beta Scope

This beta does not include account login, payments, subscriptions, advertisements, analytics, or license checks.
