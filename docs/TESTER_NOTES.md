# X75 MotionOS Tester Notes

X75 MotionOS is a free public beta for hands-free desktop control using hand gestures and optional voice commands.

## How To Start

1. Open `X75MotionOS.exe`.
2. Allow camera permission if Windows asks.
3. Press `Start`.
4. Keep your hand visible in the camera preview.
5. Use your index finger to move the cursor.
6. Press `Stop` before closing the app.

## Basic Gestures

| Gesture | Action |
| --- | --- |
| Move index finger | Move cursor |
| Thumb + index pinch | Click |
| Two quick pinches | Double click or select |
| Hold pinch | Drag |
| Index + middle fingers close together | Scroll mode |
| Move joined fingers up/down | Scroll |
| Three-finger swipe | Switch desktop |
| Four fingers | Task view |
| Zoom gesture | Zoom in/out |

## Voice Commands

Voice is disabled by default. Enable it from Advanced Settings if testing voice.

Examples:

- `open notepad`
- `search for cats`
- `open website youtube.com`
- `type hello`
- `dictate hello`
- `scroll up`
- `scroll down`
- `volume up`
- `volume down`
- `mute`
- `screenshot`
- `task view`
- `lock screen`

Shutdown, restart, and sleep voice commands are blocked unless power commands are enabled in settings.

## Beta Notes

- Good lighting improves hand tracking.
- A webcam is required.
- A microphone is optional and only needed for voice commands.
- Gesture accuracy may vary by camera, hand distance, background, and lighting.
- The beta has no login, payment, subscription, ads, or license checks.
- Camera frames are processed locally and are not intentionally recorded or uploaded.
- Voice commands may use external speech recognition if the Google voice engine is enabled.
