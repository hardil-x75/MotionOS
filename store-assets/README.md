# X75 MotionOS Store Assets

Use this folder for Microsoft Partner Center listing assets.

## Import Listing

Upload this CSV if using Partner Center listing import:

```text
StoreListing-X75MotionOS.csv
```

The CSV references these relative image paths:

```text
logos/x75-motionos-logo.png
screenshots/01-main-screen.png
screenshots/02-advanced-settings.png
screenshots/03-custom-gestures.png
screenshots/04-how-to-use-manual.png
```

If Partner Center does not accept imported images, upload the screenshots and logo manually from this same folder.

## Main Download Asset

For the package/installer section, use a direct HTTPS installer URL that does not redirect. GitHub Release links are useful for testers, but Partner Center may reject them for MSI/EXE package URLs.
